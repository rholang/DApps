Integer & Float

Inteiros

32-bits, truncados quando necessário. Reason realiza neles as operações habituais: +, -, *, /, etc.

Uso

Veja o módulo Int32 na biblioteca padrão. Para compilação de JS, consulte Js.Int.

Dicas & Truques

Cuidado quando você declarar números JavaScript! Inteiros longos podem ser truncados. Declare número JS como float em vez disso.

Floats

Float requer outros operadores: +., -., *., /., etc. Como 0.5 +. 0.6.

Uso

Nao há modulo Float na atual biblioteca padrão. Para compilação de JS, consulte Js.Int.

Decisões de Design

"Why the heck can't I just use an overloaded + for both int and float?"

There's a lot of history to this, but one day we'll be able to. Let's hang in there =)

Além disso, floats são bastante especiais em Reason/OCaml nativo. Check here if you're interested in learning some rather interesting optimizations!
