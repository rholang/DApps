Libraries

Reason Package Index（Redex） 是包的主要來源，保留了可用包的精选列表，还可搜索与浏览。。 也包含了尚未发布的包，通常表示未完成或是实验性质，可能会满足你的需求。

尚未被编入 redex 的包可能可以从 NPM 或 Github 上搜索到。 通过 BuckleScript 包装的库，习惯上会在前面加上 bs-，就像 bs-director。 这个搜索 展示了 NPM 上所有和 BS 相关的包。

如果你不能找到所需：其实为了用 JS 库，有时 你不需要一层包装 大部分的 JS 数据类型，例如 array 和 object，都清楚的与 Reason 对应。 你也可以使用熟悉的 JS API。 Be sure to also check the "Interop" and "Converting from JS" sections here!
