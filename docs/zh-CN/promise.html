Promise

Reason 通过 BuckleScript 为 JavaScript 中的 promises 提供了原生支持

  • Js.Promise.resolve: 'a => Js.Promise.t('a)
  • Js.Promise.then_: ('a => Js.Promise.t('b), Js.Promise.t('a)) => Js.Promise.t('b)
  • Js.Promise.catch: (Js.Promise.error => Js.Promise.t('a), Js.Promise.t('a)) => Js.Promise.t('a)

而在 Reason 中创建 JS promise 则使用：

  • Js.Promise.make: ( ( ~resolve: (. 'a) => unit, ~reject: (. exn) => unit ) => unit ) => Js.Promise.t('a)

类型标记的意思是， make 接受一个回调函数，该回调函数接受两个具名参数，resolve 及 reject。 两个具名参数都是 未柯里化回调函数（注意有个点）。 make 返回创建的 promise。

用法

使用管道操作符（参见 Pervasives 模块 “组合操作符”）：

let myPromise = Js.Promise.make((~resolve, ~reject) => resolve(. 2));

myPromise
|> Js.Promise.then_(value => {
     Js.log(value);
     Js.Promise.resolve(value + 2);
   })
|> Js.Promise.then_(value => {
     Js.log(value);
     Js.Promise.resolve(value + 3);
   })
|> Js.Promise.catch(err => {
     Js.log2("Failure!!", err);
     Js.Promise.resolve(-2);
   });

注意：未來可能会提供专属语法给 JS promises（async/await）。敬请关注！
