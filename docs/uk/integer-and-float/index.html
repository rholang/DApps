Integer & Float

Цілі числа

32-х бітні, усічені при необхідності. Reason надає звичайні операції для них: +, -, *, /, тощо.

Використання

Перегляньте Int32 модуль в стандартній бібліотеці. Для JavaScript - Js.Int.

Поради та трюки

** Будьте обережними, коли ви прив'язуєтеся до чисел JavaScript **! Long можуть бути усічені. Замість цього, використовуйте float.

Floats

Float включають в себе роботу з наступними операторами: +., -., *., /., etc. Наприклад, 0.5 +. 0.6.

Використання

Досі, не реалізований Float в стандартній бібліотеці. Для цього, використовуйте Js.Float.

Архітектурні рішення

"Чому я не можу просто використовувати + як для int, так і для float?"

Є багато історій на рахунок цього. Одного дня ми Вам її повідаємо =)

Крім того, плаваючі знаки є досить особливими у рідному Reason / OCaml. Check here if you're interested in learning some rather interesting optimizations!
