Imperative Loops

For 迴圈

For 迴圈從起始值（包含）開始直到結束值。

for (myBinding in startValue to endValue) {
  /* 使用 myBinding */
};

如果非必要，startValue 和 endValue 的括弧可以省略。

let xStart = 1;
let xEnd = 3;

/* 印出：1 2 3 */
for (x in xStart to xEnd) {
  print_int(x);
  print_string(" ")
};

你可以使用 downto 讓 for 迴圈反方向進行。

for (myBinding in startValue downto endValue) {
  statements
};

let xStart = 3;
let xEnd = 1;

/* 印出：3 2 1 */
for (x in xStart downto xEnd) {
  print_int(x);
  print_string(" ")
};

While 迴圈

當條件為 true 時，While 迴圈執行程式碼區塊。while 形成包括表達式及條件。

while (testCondition) {
  statements
};

提示 & 技巧

Reason 中沒有迴圈中斷的 break 關鍵字（也沒有函式內提早傳回的 return）。 然而，可以很容易的透過可變繫結來中斷 while 迴圈。

Random.self_init();

let break = ref(false);

while (! break^) {
  if (Random.int(10) === 3) {
    break := true
  } else {
    print_endline("哈嘍")
  }
};

我們對於你是否使用 map/filter/reduce 並沒有特別偏好。 某些演算法使用前者較好表達，有的是後者。
