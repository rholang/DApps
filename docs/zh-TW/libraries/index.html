Libraries

Reason 套件索引（Redex）是函式庫主要來源及繫結，保留可用套件的清單，可供搜尋及瀏覽。 這也包含了尚未發佈的套件，通常表示尚未完成或實驗性質，可能會滿足你的需求。

尚未被編入 redex 可能是無法從 NPM 或 Github 搜尋的到。 已經包裝好 BuckleScript 繫結的函式庫習慣上會在前面加上 bs-，例如 bs-director。 這個搜尋 NPM 上所有 BS 相關函式庫。

如果找不到你要的東西：有時候不一定需要繫結才能使用 JS 函式庫。 大多數的資料型別，例如 array 和物件，清楚地對應到 Reason。 你也可以存取熟悉的 JS API。 確認查過"交互作用"及"從 JS 轉換"章節！
