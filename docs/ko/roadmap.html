당신의 기여 기회

도와주고 싶으신가요?

  • Windows 에서 작업하고싶나요? 저희의 Windows 지원 도구 개선을 도와주세요!
  • 문서 및 웹사이트를 개선해주세요.
  • 문서를 번역해주세요!
  • 더 나은 오류 메시지를 제안해주세요
  • 당신이 좋아하는 에디터의 플러그인 개발을 도와주세요
  • On that note: better ocaml-language-server. This will potentially power all our editor integration in the future.
  • "re"로 시작하는 이름의 프로젝트를 더 올려주세요
  • 이미 있는 라이브러리의 래퍼를 만들어주세요! Redex를 살펴보고, 없거나 미완성인 것에 손 좀 빌려주세요! 그저 더 많은 코드 뿐만이 아니라, 더 나은 문서도 갖게 될 겁니다.
  • JavaScript에서 일반적으로 되는 프로젝트라면 무엇이든 Reason + BuckleScript 로 개조해보세요!
  • Reason과 BuckleScript에 대해 더 이야기해주세요!
  • Ping @chenglou on Forum and on Discord for more tasks.
