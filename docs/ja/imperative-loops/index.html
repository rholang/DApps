Imperative Loops

For ループ

For ループは開始値から終了値まで（終了値を含む）反復します。

for (myBinding in startValue to endValue) {
  /* use myBinding here */
};

startValue と endValue を囲む括弧は不要な場合は省略しても構いません。

let xStart = 1;
let xEnd = 3;

/* prints: 1 2 3 */
for (x in xStart to xEnd) {
  print_int(x);
  print_string(" ")
};

downto を使って、for ループカウントを逆方向にすることができます。

for (myBinding in startValue downto endValue) {
  statements
};

let xStart = 3;
let xEnd = 1;

/* prints: 3 2 1 */
for (x in xStart downto xEnd) {
  print_int(x);
  print_string(" ")
};

While ループ

While ループは、いくつかの条件が真の間コードブロックを実行します。while ループの条件部には単一の式を含みます。

while (testCondition) {
  statements
};

ヒントとコツ

Reason には loop を中断する break キーワードは（また、関数から早期 return するためのものも）ありません。 しかし、変更可能な束縛 によって、while ループから簡単に抜け出すことができます。

Random.self_init();

let break = ref(false);

while (! break^) {
  if (Random.int(10) === 3) {
    break := true
  } else {
    print_endline("hello")
  }
};

私たちは map/filter/reduce でループすべきかどうかについては問いません。 いくつかのアルゴリズムでは、前者でより良く表されますし、後者でより良く表されます。
