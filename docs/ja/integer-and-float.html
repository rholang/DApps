Integer & Float

整数

32ビット、必要に応じて切り捨てられます。Reason はそれらに一般的な操作を提供します: +, -, *, /, など。

使い方

標準ライブラリの int32 モジュールを参照してください。JS のコンパイルについては、Js.Int を参照してください。

ヒントとコツ

JavaScript の数値にバインドする時は気をつけてください！長いものは切り捨てられます。代わりに浮動小数点として JS の数値をバインドします。

浮動小数点数

浮動小数点数では他の演算子を使う必要があります: +., -., *., /., など。0.5 +. 0.6 のような。

使用法

現在の標準ライブラリには Float モジュールはありません。JS のコンパイルについては、Js.Float を参照してください。

設計方針

"Why the heck can't I just use an overloaded + for both int and float?"

There's a lot of history to this, but one day we'll be able to. Let's hang in there =)

加えて、むしろ浮動小数点数は Reason/OCaml ネイティブでは特殊です。 Check here if you're interested in learning some rather interesting optimizations!
