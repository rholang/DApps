Каррирование

JavaScriptReason
let add = a => b => a + blet add = (a, b) => a + b

Оба языка поддерживают каррирование, но в Reason каррирование встроенно и оптимизировано для избежания дополнительных аллокаций, когда это возможно.

Блоки

В Reason последовательности выражений создаются с помощью {} и возвращают последнее выражение. В JavaScript это может быть смоделировано через немедленно вызываемые функции (поскольку тело функции имеет свою собственную локальную область видимости).

</th>

<td>

let x = 23; let y = 34; x + y };

JavaScript 
let res = (function() {
  const x = 23;
  const y = 34;
  return x + y;
})();
